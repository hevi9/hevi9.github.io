Making remote git repository

To the remote server:

local> ssh remote.net
> cd /home/user/git
> mkdir project.git
> cd project.git
> git --bare init
> git config core.sharedrepository 1
> git config receive.denyNonFastforwards true

On local workstation:

> cd project
> git remote add origin ssh://user@remote.net/home/user/git/project.git
> git push -u origin master
